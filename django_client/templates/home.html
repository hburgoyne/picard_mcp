Welcome to Picard MCP

A complete memory management system built on the Model Context Protocol (MCP) standard.

About Picard MCP

Picard MCP enables users to store, retrieve, and manage their memories while controlling access permissions, and allows for semantic search and AI-powered queries based on stored memories.

Key Features

  • Secure memory storage with permission controls
  • Vector embeddings for semantic search capabilities
  • OAuth 2.0 authentication and authorization
  • Persona-based querying using stored memories
  • Memory expiration and lifecycle management
