About User Queries

This feature allows you to query a user's persona based on their memories. The response will be generated using an AI model with the user's memories as context.

  • When querying your own persona, both private and public memories will be used.
  • When querying other users' personas, only their public memories will be used.
  • Expired memories will not be included in the context.

To query your own persona, use your own user ID: {{ user.id }}
